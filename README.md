# retail-sales-forecasting

This repository uses the M5 dataset (55M+ rows) for retail sales forecasting, incorporating large-scale data processing, feature engineering, exploratory data analysis, machine learning models (Random Forest, XGBoost, LightGBM), Optuna hyperparameter optimization, model evaluation (RMSE), and feature relevance techniques including SHAP, correlation filtering, and systematic add/drop experiments to improve predictive accuracy.
